MD5

128-bit hash. Fast but not cryptographically secure. Used for checksums and non-security purposes.

SHA-1

160-bit hash. Deprecated for security. Still used in Git and legacy systems.

SHA-256

256-bit hash. Standard for cryptography. Used in Bitcoin, TLS certificates, and password storage.

SHA-512

512-bit hash. Extra security for sensitive data. Slower but more collision-resistant.
